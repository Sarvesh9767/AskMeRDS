# RefData-LLM — Investment Banking Reference Data LLM

A compact end-to-end project that demonstrates how to fine-tune a small GPT model on domain Q&A about reference data, reconciliation, and instrument mastering, and how to serve it with a simple Streamlit UI.

---
## What this repository contains

- `data/qa_data.txt` — Domain Q&A training data (Q: / A: format)
- `notebooks/fine_tuning_llm.ipynb` — Colab-compatible notebook for fine-tuning
- `model/` — Directory where the fine-tuned model will be saved (after training)
- `README.md` — This file
